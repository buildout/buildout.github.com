Buildout - software build system reloaded!

Buildout is a Python-based build system for creating, assembling and deploying applications from multiple parts, some of which may be non-Python-based. It is a coarse-grained, configuration-driven, egg-friendly and developer-oriented build system. It lets you create a buildout configuration and reproduce the same software later.
